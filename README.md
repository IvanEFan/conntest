## conntest

A simple cli tool to test your ping to a list of hostnames

### Usage

```shell
$ conntest --help
Usage: conntest [OPTIONS] COMMAND [ARGS]...

Options:
  --accurate
  --help      Show this message and exit.

Commands:
  config

$ conntest config --help
Usage: conntest config [OPTIONS] COMMAND [ARGS]...

Options:
  --help  Show this message and exit.

Commands:
  add
  clear
  remove
```
